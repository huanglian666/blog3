# 二、安装Linux

# 2.1、创建虚拟机 --- 组装一台电脑

到现在为止，相当于“组装了一台电脑”并且在光驱中放入了“Linux系统安装盘”。

# 2.2、安装CentOS

  • 在虚拟机运行界面，点左键，鼠标和键盘就会被虚拟机捕获；
  • 在当前界面下按上或下选择“Install CentOS7” --> Enter。

如何在windows下回复键盘和鼠标的使用，ctrl+alt。
